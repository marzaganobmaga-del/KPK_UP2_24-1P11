## Вариант №16. Сервис корпусов (Campus)

#### Добавление Campus

Информация требуемая для создания Campus представлена в виде таблицы:

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию | Пояснение |
|----------|----------------|-----|-------------|----------------------|-----------|
| name | Да | string | Уникальное, до 50 символов | — | Название корпуса |
| address | Да | string | Не пустое, до 200 символов | — | Адрес корпуса |
| floors | Да | integer | Больше 0 | — | Количество этажей |

Уникальные комбинации параметров: `name` (уникальное название корпуса).

Информация возвращаемая в случае удачного создания Campus:

| Параметр | Тип |
|----------|-----|
| id | integer |
| name | string |
| address | string |
| floors | integer |
| is_active | boolean |

**Примечание:** Поле `is_active` не передаётся при создании, а автоматически устанавливается в `True` и возвращается в ответе.

---

#### Изменение Campus по ID

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию | Пояснение |
|----------|----------------|-----|-------------|----------------------|-----------|
| id | Да | integer | Существует в БД | — | Идентификатор корпуса (не изменяется) |
| name | Нет | string | Уникальное, до 50 символов | — | Новое название корпуса |
| address | Нет | string | Не пустое, до 200 символов | — | Новый адрес корпуса |
| floors | Нет | integer | Больше 0 | — | Новое количество этажей |

**Примечание:** Поле `is_active` не может быть изменено через данный метод.

Информация возвращаемая в случае удачного изменения Campus:

| Параметр | Тип |
|----------|-----|
| id | integer |
| name | string |
| address | string |
| floors | integer |
| is_active | boolean |

---

#### Удаление Campus по ID
Удаление Campus по ID (мягкое удаление)

Фактически запись не удаляется из базы данных, а помечается как неактивная (устанавливается `is_active = False`).

Вернет True, если Campus была успешно деактивирована, иначе вернет False.

---

#### Получить Campus по ID

Информация возвращаемая в случае удачного поиска Campus по ID:

| Параметр | Тип | Описание | Пояснение |
|----------|-----|----------|-----------|
| id | integer | Идентификатор корпуса | Уникальный идентификатор |
| name | string | Название корпуса | Название корпуса |
| address | string | Адрес корпуса | Адрес расположения |
| floors | integer | Количество этажей | Количество этажей в корпусе |
| is_active | boolean | Статус активности | True - активен, False - неактивен |

---

#### Получить список Campus по заданным параметрам

Информация требуемая для получения списка Campus:

| Параметр | Тип | Описание | Пояснение |
|----------|-----|----------|-----------|
| min_floors | integer | Фильтр по минимальному количеству этажей | Возвращает корпуса с количеством этажей >= min_floors |
| max_floors | integer | Фильтр по максимальному количеству этажей | Возвращает корпуса с количеством этажей <= max_floors |
| exact_floors | integer | Точное количество этажей | Возвращает корпуса с точным количеством этажей |
| address_contains | string | Поиск по частичному совпадению адреса | Возвращает корпуса, адрес которых содержит указанную подстроку |
| is_active | boolean | Фильтр по статусу активности | По умолчанию True (возвращаются только активные корпуса) |

Информация возвращается в виде списка Campus:

| Параметр | Тип |
|----------|-----|
| id | integer |
| name | string |
| address | string |
| floors | integer |
| is_active | boolean |
