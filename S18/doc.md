# Вариант №18. Сервис оборудования аудиторий (Room Equipment Service)

## Добавить оборудование

### Информация, требуемая для создания оборудования

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию | Пояснение |
|----------|---------------|-----|-------------|----------------------|-----------|
| name | Обязательно | Строка | 1..100 символов | – | Название оборудования |
| type_id | Обязательно | Целое | > 0, ссылка на id в таблице equipment_types | – | Тип оборудования |
| room_id | Обязательно | Целое | > 0 | – | ID аудитории |
| status_id | Необязательно | Целое | > 0, ссылка на id в таблице equipment_statuses | 1 (active) | Статус оборудования |
| inventory_number | Необязательно | Строка | Уникальный, до 50 символов | NULL | Инвентарный номер |
| description | Необязательно | Строка | До 500 символов | "" | Описание |

**Уникальные комбинации параметров:** 
- `inventory_number` (уникальный)
- `room_id` + `name` (в одной аудитории не может быть двух одинаковых названий оборудования)

### Выходные данные

| Параметр | Тип |
|----------|-----|
| id | Целое |
| name | Строка |
| type_id | Целое |
| room_id | Целое |
| status_id | Целое |
| inventory_number | Строка |
| description | Строка |
| is_active | Булево |

## Дополнительные таблицы

### Таблица equipment_statuses

| Параметр | Тип | Ограничение | Значение по умолчанию |
|----------|-----|-------------|----------------------|
| id | Целое | PRIMARY KEY, AUTO_INCREMENT | – |
| name | Строка | Уникальный, 1..50 символов | – |
| description | Строка | До 200 символов | "" |

**Начальные данные:**
| id | name | description |
|----|------|-------------|
| 1 | active | Оборудование работает исправно |
| 2 | broken | Оборудование сломано, требует ремонта |
| 3 | maintenance | Оборудование на обслуживании/ремонте |

### Таблица equipment_types

| Параметр | Тип | Ограничение | Значение по умолчанию |
|----------|-----|-------------|----------------------|
| id | Целое | PRIMARY KEY, AUTO_INCREMENT | – |
| name | Строка | Уникальный, 1..100 символов | – |

**Начальные данные:**
| id | name |
|----|------|
| 1 | projector |
| 2 | computers |
| 3 | machines |
| 4 | boards |
| 5 | other |
